ℹ️ Expie Maker Manual

Welcome to Expie Maker, a browser-based custom skin and pixel-art creator for Casualties: Unknown. Below is a guide to everything you can do in this studio!

🎨 Core Drawing Tools

  • B Pencil: Standard pixel drawing brush.
  • E Eraser: Erases canvas pixels.
  • G Fill Bucket: Fills connected areas of the same color.
  • R Color Replace: Swap a specific color across the active part.
    • Tolerance: Match shades with variable color-distance.
    • Preserve Shading: Keeps relative white-to-darkness gradients when coloring!
  • I Eyedropper: Samples any color on the active canvas.

🎛️ Drawing Modifiers

  • 🪞 Horizontal Mirror (Symmetry):

    When active, paints and erases symmetrically across the center. Renders a stunning vertical dashed blue center axis line on the canvas.

  • 🔄 Canvas Flipping & Transforms:

    Quickly flip your active part to adjust layout directions. Shortcuts:

    • H Flip active canvas Horizontally.
    • V Flip active canvas Vertically.
  • Brush Opacity (0% - 100%):

    Controls brush and fill transparency, allowing custom shading overlays, gradients, and soft-blending effects.

  • Saved Color Swatches:

    Add, pick, and persist your custom color palettes. Custom swatches are automatically saved locally.

💾 Auto-Saves & Snapshot Library

  • Real-time Auto-Saving:

    Your active skin, character name, preview selections, and custom swatches are continuously backed up in your browser.

  • Ctrl+Z / Ctrl+Y History:

    Your full undo/redo action states are serialized and saved so you never lose your history on refresh.

  • Skins Library & Snapshots:

    Create entirely separate skins, view exact file size metrics, capture manual snapshots, and configure background auto-snapshots every few minutes.

📂 Asset Import & Game Export

  • Batch Folder Import:

    Select multiple PNGs. The creator scans the filenames (e.g., experimentEyeOpen.png, TailAnim_0.png) and assigns them to the correct parts instantly.

  • Single Part Import:

    Quickly overwrite the selected sprite part with any custom PNG file of matching size.

  • Export ZIP:

    Downloads a packaged, clean character ZIP archive matching the required folder layout for Casualties: Unknown.

🖼️ Reference Image Overlay

  • Tracing Paper Reference:

    Load any reference sketch, character sheet, or template PNG/JPG to display as a semi-transparent tracing overlay beneath your grid lines.

  • Positioning & Navigation Shortcuts:

    Move and size your reference overlay directly on the canvas without clicking inputs:

    • 🖱️ Right-drag (or Alt + Drag) to pan/move overlay.
    • 🛞 Alt + Scroll (or Shift + Scroll) to scale/size overlay.
  • Filters & Styling:

    Open the overlay settings popup (⚙️) to adjust Fit Modes (Stretch, Contain, Cover, Original), invert/grayscale colors, or apply a solid color tint overlay for high-contrast tracing.
